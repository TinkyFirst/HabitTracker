## 🎨 Кольорова Схема

### Градієнти:
- **Прогрес:** 🔵 Синій → 🟣 Фіолетовий
- **Стріки:** 🟠 Помаранчевий
- **Виконано:** 🟢 Зелений чекмарк
- **Не виконано:** ⚪️ Сірий кружечок

### Фон:
- **Home Screen:** Прозорий з легким блюром
- **Lock Screen:** Адаптується під системний дизайн
- **StandBy:** Контрастний, легко читається

---

## 📏 Розміри Віджетів

### Home Screen:
- **Small:** 2x2 клітинки (~158x158 pt)
- **Medium:** 4x2 клітинки (~329x158 pt)
- **Large:** 4x4 клітинки (~329x345 pt)

### Lock Screen:
- **Circular:** Кругле (~68x68 pt)
- **Rectangular:** Прямокутне (~157x68 pt)
- **Inline:** Один рядок тексту

### StandBy:
- Використовує той самий розмір як Home Screen
- Але оптимізовані для горизонтального перегляду

## 💡 Поради по Дизайну

### Do's ✅
- Використовуйте Interactive віджети для швидкого доступу
- Розміщуйте найважливіші віджети на першому екрані
- Комбінуйте різні розміри для балансу
- На Lock Screen використовуйте компактні віджети

### Don'ts ❌
- Не розміщуйте занадто багато великих віджетів
- Не дублюйте однакову інформацію
- Не перевантажуйте StandBy Mode віджетами
