## 📁 Як знайти Project Navigator (ліва панель)

**Якщо не видно:**
- Натисни **⌘+1** (Command + 1)
- Або View → Navigators → Project

**Що тут:**
```
📁 TrackHabit (синя іконка) ← Це корінь проекту, натискай для налаштувань
  ├─ 📁 TrackHabit (жовта папка) ← Папка з файлами
  │   ├─ TrackHabitApp.swift
  │   ├─ LanguageManager.swift
  │   ├─ FirebaseNotificationManager.swift
  │   └─ GoogleService-Info.plist ← Сюди додаємо файл!
  ├─ 📁 Assets.xcassets
  ├─ 📁 Products
  └─ 📦 Package Dependencies ← Тут Firebase після додавання
```

## ❓ Не можу знайти щось?

### Панелі не видно?

```
View → Navigators → Show Project Navigator    (⌘+1)
View → Inspectors → Show File Inspector       (⌘+⌥+1)
View → Debug Area → Show Debug Area           (⌘+⇧+C)
```

### Кнопка + Capability не знаходжу?

1. Точно вибрано **TARGET** (не Project)?
2. Точно на табі **Signing & Capabilities**?
3. Кнопка + Capability має бути **зліва вгорі** під табами

### Файл не додається в проект?

1. Перевір що тягнеш в **жовту папку** (не синю іконку)
2. В діалозі обов'язково ✅ **Copy items if needed**
3. В діалозі обов'язково ✅ **TrackHabit** в targets
